# 💫 About Me

I'm a data professional passionate about transforming raw data into scalable solutions and actionable business insights.

My experience spans **Data Engineering, Data Analytics, Business Intelligence, Machine Learning, and Generative AI**, allowing me to work across the complete data lifecycle—from data ingestion and transformation to visualization, predictive modeling, and AI-powered applications.
